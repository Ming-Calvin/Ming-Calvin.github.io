一、CSS概述

1. css语法规范

CSS 规则由两个主要的部分构成：选择器以及一条或多条声明。

  • 选择器是用于指定 CSS 样式的 HTML 标签

  • 花括号内是对该对象设置的具体样式

  • 属性和属性值以“键值对”的形式出现

  • 属性是对指定的对象设置的样式属性，例如字体大小、文本颜色等

  • 属性和属性值之间用英文“:”分开

  • 多个“键值对”之间用英文“;”进行区分

2．代码分格

A．样式格式书写（展开式）

B．样式大小写（样式选择器，属性名，属性值关键字全部使用小写字母）

C. 空格规范

注：

  • 属性值前面，冒号后面，保留一个空格

  • 选择器（标签）和大括号中间保留空格
    二、Emmet语法

1. 快速生成HTML结构语法

  • 生成标签直接输入标签名，按tab键。（ 比如 div 然后tab 键， 就可以生成
    <div></div> ）

  • 生成多个相同标签，加上 * 号。（ 比如 div*3 就可以快速生成3个div ）

  • 生成父子级关系的标签，用 > 号。（ 比如 ul > li ）

  • 生成兄弟关系的标签，用 + 号。（ 比如 div+p ）

  • 生成带有类名或者id名的标签，用 .类名 、#id名（ 默认生成div标签 ） 或者
    标签名.类名 、标签名#id名，按tab键 （ 比如 p.one ）

  • 生成的div类名是有顺序的，用自增符号 $ （ 比如 .demo$*5 ）

  • 生成的标签内部写内容可以用 { } 表示

2. 快速生成CSS样式语法

CSS采取简写形式：

  • 比如 w200 按tab 可以 生成 width: 200px;

  • 比如 lh26px 按tab 可以生成 line-height: 26px;

三、基础选择器

1. CSS选择器的作用

作用：根据不同需求选择不同的标签

2. 选择器分类

基础选择器

选择器

复合选择器

3. 基础选择器

由单个选择器组成，包括：

4. 标签选择器

使用HTML标签名称作为选择器，为页面中某一类标签指定统一的 CSS 样式

  1. 标签名{

  2. 属性1: 属性值1;

  3. 属性2: 属性值2;

  4. 属性3: 属性值3;

  5. …

  6. }

作用：把某一类标签全部选择出来

优点：能快速为页面中同类型的标签统一设置样式

缺点：不能设计差异化样式，只能选择全部的当前标签
5. 类选择器

用于差异化选择不同的标签，可以单独选一个或者某几个标签。

  • 在 CSS 中，类选择器以一个点“.”号显示

  1. .类名 {

  2. 属性1: 属性值1;

  3. …

  4. }

  • 在 HTML 中以 class 属性表示
  1. <div class=’red’> 变红色 </div>

注意：

  • 类选择器使用“.”进行标识，后面紧跟类名

  • 长名称或词组可以使用中横线来为选择器命名。

  • 不要使用纯数字、中文等命名，尽量使用英文字母来表示。

6. 类选择器-多类名

用于给一个标签指定多个类名，这些类名都可以选出这个标签

  1. <style>

  2. .red {

  3. color: red;

  4. }

  5. .font35{

  6. font-size: 35px;

  7. }

  8. </style>

  9. <body>

  10. <div class=”red font35”> 小明是男神 </div>

  11. </body>

注意：

  • 在标签class 属性中写多个类名

  • 多个类名中间必须用空格分开

使用场景：

  • 把公共部分放在一个类里

  • 标签可以先调用公共的类再调用单独的类

7. id选择器

  1. #id名 {

  2. 属性1: 属性值1;

  3. …

  4. }

id 选择器和类选择器的区别：

  • 类选择器可以多次使用，id选择器只能调用一次

  • 类选择器主要用于修改样式

  • id 选择器一般用于页面中唯一性的元素（和JavaScript搭配使用）

8.通配符选择器

使用“*”定义，用于选取页面中所有元素（标签）。

  1. * {

  2. 属性1: 属性值1;

  3. …

  4. }

四、复合选择器

1. 概述

复合选择器是由两个或多个基础选择器，通过不同的方式组合而成的

分类

2. 后代选择器 (重要）

后代选择器可以选择父元素里面子元素。

  1. /*把ol的li选出来改成pink色 */

  2. ol li {

  3. color: pink;

  4. }

注：

  • 元素2只要是元素1 的后代即可

  • 元素1是父级，元素2是子级，最终选择的是元素2

  • 元素1和元素2可以是任意基础选择器

  1. .nav li a {

  2. color: yellow;

  3. }

3. 子选择器 (重要）

子元素选择器只能选择作为某元素的最近一级子元素

  1. <style>

  2. .nav > a {

  3. color: red;

  4. }

  5. </style>

4. 并集选择器 (重要）

并集选择器可以选择多组标签, 同时为他们定义相同的样式

  1. div,

  2. p,

  3. .pig li {

  4. color: pink;

  5. }

注意：

  • 并集选择器喜欢竖着写

  • 最后一个选择器不需要加逗号

5. 伪类选择器

伪类选择器用于向某些选择器添加特殊的效果（常用链接伪类选择器）

  1. a:hover {

  2. color: red; /* 鼠标经过的时候，由原来的 灰色 变成了红色 */

  3. }

6. 链接伪类选择器

A． a:link 选择未被访问的链接

  1. a:link {

  2. color: #333;

  3. text-decoration: none;

  4. }

B． a:visited 选择已被访问过的链接

  1. a:visited {

  2. color: orange;

  3. }

C． a:hover 选择鼠标经过时的链接

  1. a:hover {

  2. color: skyblue;

  3. }

D． a:active 选择鼠标按下未弹起的链接

  1. a:active {

  2. color: green;

  3. }

注意：按照LVHA的循顺序声明 :link － :visited － :hover － :active

7. :focus 伪类选择器

:focus 伪类选择器用于选取获得焦点(光标)的表单元素。（ <input>
类表单元素才能获取 ）

  1. input:focus {

  2. background-color: pink;

  3. color: red;

  4. }

五、元素显示模式

1. 概述

元素的显示模式就是元素（标签）以什么方式来显示，根据元素显示模式的不同元素标签被分为了两类:行内元素、块级元素

2. 块元素

常见的块元素有 <h1> …
<h6>、<p>、<div>、<ul>、<ol>、<li>等，<div> 标签是最典型的块元素。

特点：

  • 独占一行

  • 高度，宽度、外边距以及内边距都可以控制。

  • 宽度默认是容器（父级宽度）的100%。

  • 是一个容器及盒子，里面可以放行内或者块级元素。

注意：文字类的元素（ <p> 、<h1> … <h6> ）内不能使用块级元素

3.行内元素

常见的行内元素有
<a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span>等，<span>
标签是最典型的行内元素。

特点：

  • 相邻行内元素在一行上，一行可以显示多个

  • 高、宽直接设置是无效的

  • 默认宽度是它本身内容的宽度

  • 行内元素只能容纳文本或其他行内元素

注意：链接里面不能再放链接，链接 <a> 里面可以放块级元素（转换块级模式）

4. 行内块元素

行内块元素同时具有块元素和行内元素的特点，常见标签：<img/>、<input/>、<td>

特点：

  • 和相邻行内元素（行内块）在一行上，但是之间会有空白缝隙

  • 一行可以显示多个（行内元素特点）

  • 默认宽度就是它本身内容的宽度（行内元素特点）。

  • 高度，行高、外边距以及内边距都可以控制（块级元素特点）

5. 元素显示模式总结

6. 元素显示模式转换

一个模式的元素需要另外一种模式的特性

方法：

  • 转换为块元素：display:block;

  1. a {

  2. width: 150px;

  3. height: 50px;

  4. background-color: pink;

  5. /* 把行内元素 a 转换为 块级元素 */

  6. display: block;

  7. }

  • 转换为行内元素：display:inline;

  1. div {

  2. background-color: purple;

  3. /* 把 div 块级元素转换为行内元素 */

  4. display: inline;

  5. }

  • 转换为行内块：display: inline-block;

  1. span {

  2. width: 300px;

  3. height: 30px;

  4. background-color: skyblue;

  5. display: inline-block;

  6. }

7. 单行文字垂直居中

解决方案：让文字的行高等于盒子的高度

原理：

行高的上空隙和下空隙把文字挤到中间了.
是如果行高小于盒子高度,文字会偏上,如果行高大于盒子高度,则文字偏下

七、CSS背景

1. 背景颜色

background-color 属性定义了元素的背景颜色，一般情况下元素背景颜色默认值是
transparent（透明）

  1. div {

  2. width: 200px;

  3. height: 200px;

  4. /* background-color: transparent;*/

  5. background-color: pink;

  6. }

2. 背景图片

background-image 属性描述了元素的背景图像。常见于 logo
或者一些装饰性的小图片或者是超大的背景图片,非常便于控制位置。（精灵图）

![0HG4]D7TTNURPUGGAQBUB2G](media/8bcec6284e3b4a96f14aaa76612e5743.png)

3. 背景平铺

background-repeat 属性可以控制背景图像的平铺

![CYG)A6T08Y4{$`_A}AFVND

  1. div {

  2. width: 300px;

  3. height: 300px;

  4. background-color: pink;

  5. background-image: url(images/logo.png);

  • 背景图片不平铺
  1. background-repeat: no-repeat;
  • 默认的情况下,背景图片是平铺的
  1. background-repeat: repeat;
  • 沿着x轴平铺
  1. background-repeat: repeat-x;
  • 沿着Y轴平铺
  1. background-repeat: repeat-y;

注：页面元素既可以添加背景颜色也可以添加背景图片 只不过背景图片会压住背景颜色

4. 背景图片位置

background-position 属性可以改变图片在背景中的位置

参数：x 坐标和 y 坐标（方位名词 或者 精确单位）

![FP]CGHU6NBY@%HRLV9Z]L9J](media/ea64b64c4687d56dd83142a2e62e0ceb.png)

注：

A．参数是方位名词

  • 如果指定的两个值都是方位名词，则两个值前后顺序无关 

  • 如果只指定了一个方位名词，另一个值省略，则第二个值默认居中对齐

B. 参数是精确单位

  • 如果参数值是精确坐标，那么第一个肯定是 x 坐标，第二个一定是 y 坐标

  • 如果只指定一个数值，那该数值一定是 x 坐标，另一个默认垂直居中

C．参数是混合单位

 如果指定的两个值是精确单位和方位名词混合使用，则第一个值是 x 坐标，第二个值是
y 坐标

5. 背景图像固定（背景附着）

background-attachment 属性设置背景图像是否固定或者随着页面的其余部分滚动

![YMC5E}YOCJDGRWVJT8Q5YF

6. 背景复合写法

background: 背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置

7. 背景色半透明

注意：

  • 最后一个参数是alpha透明度，取值范围在 0~1之间

  • 习惯把0.3的0省略掉，写为background: rgba(0, 0, 0, .3);

  • 背景半透明是指盒子背景半透明，盒子里面的内容不受影响

  1. div {

  2. width: 300px;

  3. height: 300px;

  4. /* background: rgba(0, 0, 0, 0.3); */

  5. background: rgba(0, 0, 0, .3);

  6. }
